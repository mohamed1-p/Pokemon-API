# Pokemon-API
###### Is a RESTful API for reviewing pokemons
# Pokemon-API developed using 
###### Spring Boot
###### Spring security
###### Spring JPA
###### MySQL Database.

# Following features
###### Add Pokemon
###### Edit Pokemon
###### List Pokemons
###### Delete Pokemon
###### Add Review for a pokemon
###### Edit Review for a pokemon
###### List reviews of a pokemon
###### Delete a pokemon review
###### Login
# Technology stack
###### Java 8+
###### Spring Boot
###### MySQL database
###### Eclipse IDE
###### Tomcat server 10
